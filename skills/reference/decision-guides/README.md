# Reference: Decision Guides

Structured guides for making configuration choices on HPE hardware. Use these when you need to select the right option for a specific scenario rather than just knowing the syntax.

## Guides

| Guide | Description | Use When |
|-------|-------------|----------|
| [raid-selection.md](raid-selection.md) | RAID level comparison with usable capacity, fault tolerance, and performance characteristics | Choosing RAID level for a new array or advising on storage layout |
| [nic-configuration.md](nic-configuration.md) | iLO NIC selection, teaming/bonding modes, and network boot method selection | Configuring management networking or selecting a PXE/HTTP boot method |
| [firmware-strategy.md](firmware-strategy.md) | Online vs offline update approaches, update frequency, rolling vs batch, and pre-update checklist | Planning a firmware update cycle or responding to a security advisory |
